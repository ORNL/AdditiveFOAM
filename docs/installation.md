# Installation

## Prerequisites

Make sure you have the following installed:

- Git
- CMake
- GCC
- OpenFOAM

## Steps

1. Clone the repository:
    ```bash
    git clone https://github.com/ORNL/AdditiveFOAM.git
    ```

2. Navigate to the project directory:
    ```bash
    cd AdditiveFOAM
    ```

3. Build the project:
    ```bash
    mkdir build && cd build
    cmake ..
    make
    ```
